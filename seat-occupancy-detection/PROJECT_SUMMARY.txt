TROUBLESHOOTING
================================================================================

Issue: Webcam not working
Fix: Check permissions, test with test_setup.py

Issue: GPU not detected
Fix: Set DEVICE='cpu' in config.py or install CUDA

Issue: Poor detection accuracy
Fix: Improve lighting, recalibrate zones, adjust threshold

Issue: Dependencies error
Fix: Reinstall with pip install -r requirements.txt

See README.md Troubleshooting section for more details.
